# Demo Videos

This page catalogs the MP4 demos under `assets/`. The files are H.264 MP4s and
are meant to be uploaded to GitHub as video attachments when they need to render
inline in a README or release page.

Each video is embedded below in sequence.


## Code View

`code_view_2x_h264.mp4` shows the docked code panel working with the diagram:
open source for a selected module, type, field, method, or function, highlight
the defining span, and click source lines to navigate the diagram back to the
matching element.

https://github.com/user-attachments/assets/0bcfb92a-f4b3-444c-b3bc-2735fd27dca9


## Diff View

`diff_view_1_5x_h264.mp4` shows diff mode: changed entities carry add/delete or
modified markers in the architecture view, while the code panel renders the
same file as an inline diff with old and new line coordinates.

https://github.com/user-attachments/assets/51262599-51b9-4626-9a8e-f4d06923ce0d


## Overview

`overview_h264.mp4` shows the Space-key overview toggle: press Space to fit
every diagram object on screen, then press Space again to return to the working
view around the current cursor or focus point.

https://github.com/user-attachments/assets/b487338c-1964-4a67-9a2c-2118af341a3b

## Diff Walkthrough

`diff_walkthrough_h264.mp4` shows an agent-posted review tour in diff mode. The
tour bubble and step list advance through anchored observations while the
diagram focuses the relevant elements or verified arrows and the code panel
keeps the diff visible when it is open.

https://github.com/user-attachments/assets/ce56ea8a-da3e-4c45-9569-98ac9eea8c7b


## Planning Walkthrough

`planning_walkthrough_h264.mp4` shows a planning tour: longer markdown-rich
steps describe phases of a change, and each non-text step is anchored to the
source location the agent expects the work to touch.

https://github.com/user-attachments/assets/3af4eac1-a1ad-4684-be4e-38520b40713f


## Caller And Callee

`caller_callee.mp4` shows caller-to-callee navigation. Callable rows expose
outgoing and incoming call markers; selecting one reveals the concrete
caller -> callee edge and can pan to the other endpoint.

https://github.com/user-attachments/assets/881c4862-b029-4994-bbbb-f84cbbf37574

## Ownership Walkthrough

`ownership_walk_1_5x_h264.mp4` shows structural ownership arrows between types.
Field and variant payload relationships can reveal owner -> owned edges, and
the viewer keeps the clicked row or type anchored while the related endpoint is
expanded into view.

https://github.com/user-attachments/assets/aad017f9-6abf-43f2-a4a3-e11b3e4b720b
