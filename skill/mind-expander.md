---
name: mind-expander
description: |
  mind-expander opens a live, interactive diagram of the user's
  codebase in their browser — a shared visual workspace where you pin
  narrated steps to exact file:line locations. Use it for codebase
  walkthroughs, architectural planning, code review, or any
  explanation that benefits from seeing structure visually. Trigger
  words: "tour", "walk me through", "show me on the diagram",
  "visualize this". Before first use in a session, run
  `mind-expander --help` and capture the full output (~300 lines) —
  it contains the complete protocol reference.
---
